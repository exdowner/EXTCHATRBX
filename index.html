# Xeno Firebase Voice Prototype ## index.html (GitHub Pages) ```html Xeno Voice
VOICE OFF
``` --- # Roblox/Xeno Script Addon Adiciona isso no teu script: ```lua local VoiceHelperURL = "https://TEUUSUARIO.github.io/voice" local VoiceEnabled = false local VoiceButton = Instance.new("TextButton") VoiceButton.Size = UDim2.new(0,32,0,32) VoiceButton.Position = UDim2.new(1,-114,1,-42) VoiceButton.BackgroundColor3 = Color3.fromRGB(55,55,55) VoiceButton.Text = "🎤" VoiceButton.TextSize = 18 VoiceButton.TextColor3 = Color3.new(1,1,1) VoiceButton.Parent = ChatWindow local VoiceCorner = Instance.new("UICorner") VoiceCorner.CornerRadius = UDim.new(0,6) VoiceCorner.Parent = VoiceButton local VoiceIndicator = Instance.new("TextLabel") VoiceIndicator.Size = UDim2.new(0,120,0,20) VoiceIndicator.Position = UDim2.new(1,-240,1,-38) VoiceIndicator.BackgroundTransparency = 1 VoiceIndicator.Text = "VOICE: OFF" VoiceIndicator.TextColor3 = Color3.fromRGB(255,80,80) VoiceIndicator.Font = Enum.Font.SourceSansBold VoiceIndicator.TextSize = 16 VoiceIndicator.Parent = ChatWindow ``` --- # Toggle Voice ```lua VoiceButton.MouseButton1Click:Connect(function() VoiceEnabled = not VoiceEnabled if VoiceEnabled then VoiceButton.BackgroundColor3 = Color3.fromRGB(0,170,90) VoiceIndicator.Text = "VOICE: ON" VoiceIndicator.TextColor3 = Color3.fromRGB(0,255,120) InserirMensagemLocal( "Voice", "Voice Chat ativado.", "rgb(0,255,120)" ) pcall(function() if setclipboard then setclipboard(VoiceHelperURL) end end) if identifyexecutor then print("Open this URL in browser:", VoiceHelperURL) end else VoiceButton.BackgroundColor3 = Color3.fromRGB(55,55,55) VoiceIndicator.Text = "VOICE: OFF" VoiceIndicator.TextColor3 = Color3.fromRGB(255,80,80) InserirMensagemLocal( "Voice", "Voice Chat desativado.", "rgb(255,80,80)" ) end end) ``` --- # Como usar 1. Hospeda o HTML no GitHub Pages 2. Abre a página no navegador 3. Permite acesso ao microfone 4. Executa o script no Xeno 5. Liga o botão 🎤 6. Todo mundo usando a mesma página/Firebase escuta os chunks --- # Observações * Isso funciona mais como walkie-talkie/chunks do que call perfeita. * Firebase Realtime Database tem delay. * Quanto menor os chunks, melhor. * Tu pode adicionar: * canais * salas * mute * proximity fake * lista de usuários * global/local * push-to-talk
